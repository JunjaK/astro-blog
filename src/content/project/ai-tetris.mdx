## 설명
Perplexity AI의 코드 생성 기능을 활용하여 딸각 한 번에 만든 Nuxt 3 기반 테트리스 게임입니다.  
Nuxt UI 4와 Tailwind CSS로 스타일링하였으며, SSG로 배포된 클라이언트 사이드 게임입니다.  
AI가 생성한 코드만으로 완성된 프로젝트로 역시 쌓인 데이터가 많은 테트리스 같은 거는 AI가 쉽게 쉽게 잘 만드는 구나 싶었습니다.

## 기술 스택
- Nuxt 3: SSG 프레임워크
- Nuxt UI 4: UI 컴포넌트
- Tailwind CSS: 스타일링
- Netlify: 정적 사이트 배포

## 주요 기능
- 블록 이동, 회전, 소프트/하드 드롭
- 블록 보관 (Hold) 및 다음 블록 미리보기 (Next)
- 점수, 라인, 레벨 시스템
- 일시정지 기능
- 다크 테마 UI
